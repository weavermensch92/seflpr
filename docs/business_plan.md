# SelfPR — 사업계획서

**서비스명:** SelfPR
**버전:** 1.0
**작성일:** 2026년 3월
**분류:** AI SaaS / HR Tech / EdTech

---

## 1. 서비스 요약 (Executive Summary)

SelfPR은 한국 취업 준비생을 위한 **AI 기반 자기소개서 자동 작성 + 면접 예상 질문 연습 플랫폼**이다.

기업명과 지원 포지션을 입력하면 AI가 해당 기업을 직접 리서치하고, 유저의 학력·경력·프로젝트 데이터를 조합하여 기업 방향에 맞는 스토리텔링 자기소개서를 자동 생성한다. 생성된 자소서를 기반으로 HR + 기술 담당자 이중 시각의 면접 예상 질문을 출제하고, 유저의 답변에 대해 실시간 피드백을 제공하는 1문 1답 연습 기능까지 제공한다.

**핵심 가치:** 단순 글쓰기 도구가 아닌, 기업을 스스로 조사하고 판단하는 **능동적 AI 취업 코치**.

## 2. 문제 정의

### 2-1. 현재 취업 준비의 문제점

**자기소개서 작성**
- 취업 준비생 1인당 평균 12개 기업에 지원 (2025 취업포털 설문)
- 기업마다 다른 인재상, 포지션별 다른 요구사항 → 각 자소서에 맞춤화 필요
- 맞춤화된 자소서 1개 작성에 평균 4-8시간 소요
- ChatGPT로 작성 시 AI 어투가 그대로 드러나 감점 요인

**면접 준비**
- 자소서 기반의 맞춤형 예상 질문 없음 → 일반적인 면접 질문집에 의존
- 혼자 연습할 때 피드백 수단 없음
- 꼬리 질문 대응 훈련 어려움 → 실전에서 당황

**개인정보 보안**
- 기존 자소서 작성 도구는 학력·경력 등 민감 정보를 서버에 무단 저장
- 유저는 자신의 개인정보가 어디에 저장되는지 모름

### 2-2. 시장 규모

| 구분 | 수치 | 출처 |
|------|------|------|
| 연간 취업 지원 건수 (한국) | 약 **800만 건** | 통계청 2025 |
| 취업 준비생 수 | 약 **320만 명** | 통계청 2025 |
| HR Tech 국내 시장 | **약 2조 원** (2025) | KDB미래전략연구소 |
| AI 글쓰기 도구 글로벌 시장 | **약 2.5조 원** (2025) | Grand View Research |
| 연 성장률 | **27%** CAGR | Grand View Research |

### 2-3. 타깃 유저

**Primary:** 대학교 3-4학년 및 졸업 후 1-2년차 취업 준비생
- 연령대: 22-27세
- 특징: 디지털 네이티브, AI 도구 친숙, 취업 압박 높음
- 규모: 약 150만 명 (활성 취업 준비층)

**Secondary:** 이직을 준비하는 직장인 (3-7년차)
- 연령대: 27-35세
- 특징: 포지션별 맞춤 자소서 필요, 시간 부족
- 규모: 약 50만 명

## 3. 솔루션

### 3-1. 핵심 기능

#### A. AI 자기소개서 자동 생성

```
기업명 + 포지션 입력
→ AI 기업 리서치 (뉴스/공식사이트/합격사례 자동 수집)
→ 유저 프로필과 기업 요구 매칭
→ STAR 기법 스토리텔링 자소서 초안 생성
→ AI 리뷰 + 어투 검토 + 갭 분석
→ PDF/Word 내보내기
```

**차별점:**
- 기업 리서치를 AI가 직접 수행 → 유저가 조사할 필요 없음
- 자소서를 쓰는 AI가 20년차 HR + 10년차 실무자 관점으로 작성
- AI 어투 감지 → 사람이 쓴 것처럼 자동 재작성

#### B. 면접 예상 질문 연습

```
자소서 최신 버전 자동 스냅샷
→ 기업 가치관 + 면접 레퍼런스 서치
→ 1문 1답 출제 (자소서/기업가치관/직무/상황 4가지 유형)
→ 유저 답변 → 실시간 AI 피드백 (SSE 스트리밍)
→ 꼬리 질문 or 신규 질문 유저 선택
→ 세션 종료 → 취약 영역 분석 + PDF 다운로드
```

**차별점:**
- 자소서 내용 기반 맞춤형 질문 (일반적인 질문집과 다름)
- 답변을 직접 주지 않고 "대응력을 기르는" 피드백 방식
- 꼬리 질문 훈련 → 실전 면접 압박 상황 대비

#### C. 개인정보 보호 우선 설계

- **파일 원본 서버 미저장**: 업로드 파일은 메모리에서만 처리 후 즉시 폐기
- **미결제 시 로컬 저장**: 포인트 충전 전까지 프로필 데이터는 브라우저 localStorage만
- **투명한 고지**: "이 데이터는 현재 이 브라우저에만 저장됩니다" 항상 표시

### 3-2. 서비스 플로우

```
[회원가입]
    ↓
[내 프로필 등록]
  - 파일 업로드 (PDF/Excel/Word/이미지)
  - 수동 입력 또는 링크 파싱
  - 결제 전 → localStorage만 저장
    ↓
[포인트 충전] (10,000원 = 100P)
    ↓
[자소서 프로젝트 생성] (50P)
  - 기업명 + 포지션 + 질문 입력
  - AI 자동 생성 → 에디터에서 수정
    ↓
[면접 연습 시작] (60P)
  - 1문 1답 → 피드백 → 꼬리/신규 질문 선택
  - 세션 요약 + 취약 분석
```

## 4. 비즈니스 모델

### 4-1. 포인트 과금 구조

| 항목 | 포인트 | 원화 환산 |
|------|--------|---------|
| 포인트 충전 | 100P / 10,000원 | 1P = 100원 |
| 자소서 프로젝트 생성 | **50P** | 5,000원 |
| 면접 연습 세션 시작 | **60P** | 6,000원 |
| 면접 신규 질문 | **3P** | 300원 |
| 면접 꼬리 질문 | **1P** (질문당 최대 5개) | 100원 |

### 4-2. AI 원가 분석

**자소서 생성 1건당 AI 비용**

| 단계 | 모델 | 비용 |
|------|------|------|
| 기업 리서치 | GPT-4o-mini | ~$0.003 |
| 기업 분석 | Claude Sonnet 4.6 | ~$0.05 |
| 프로필 매칭 | GPT-4o-mini | ~$0.003 |
| 자소서 작성 (6문항) | Claude Sonnet 4.6 | ~$0.07 |
| **합계** | | **~$0.13 (약 180원)** |

**수익 = 5,000원 - 180원 = 4,820원/건 (마진율 96%)**

**면접 연습 세션당 AI 비용 (20문답 기준)**

| 항목 | 비용 |
|------|------|
| 시스템 프롬프트 + 첫 질문 | ~$0.03 |
| 답변 피드백 × 20 | ~$0.40 |
| 신규 질문 × 10 | ~$0.15 |
| 꼬리 질문 × 10 | ~$0.10 |
| **합계** | **~$0.68 (약 940원)** |

**세션 수입 = 60P(6,000원) + 신규10×3P(3,000원) + 꼬리10×1P(1,000원) = 100P (10,000원)**
**수익 = 10,000원 - 940원 = 9,060원/세션 (마진율 91%)**

### 4-3. 기업 리서치 캐시로 추가 절감

- 동일 기업+포지션 2번째 요청부터 캐시 재활용 → AI 비용 0
- 인기 기업(삼성전자, SK하이닉스 등)은 수십 명이 공유 → 건당 원가 급감
- 캐시 7일 TTL → 최신성 유지

### 9-2. 손익분기점

```
월 고정비: 40만원
평균 건당 마진: 자소서 4,820원 / 면접 14,060원

손익분기 = 40만원 / 평균 5,000원
           = 월 80건 유료 거래
           ≈ 월 유료 유저 40명 (1인 2건 기준)
```

**손익분기점 매우 낮음 → 초기 자금 리스크 최소화**

## 12. 결론

SelfPR은 **취업 준비생의 가장 큰 고통점** — 맞춤형 자소서 작성과 면접 준비 — 을 AI로 해결하는 서비스다.

단순한 ChatGPT 래퍼가 아닌, **기업을 스스로 조사하고, 유저 프로필과 매칭하고, 면접까지 코치하는** 완성형 AI 취업 코치다.

개인정보를 최대한 보호하는 로컬 우선 설계, 포인트 기반의 유연한 과금, 어드민에서 프롬프트를 실시간 개선하는 운영 구조는 **빠른 제품 개선과 높은 마진율**을 동시에 달성하게 한다.

**한국 취업 준비생 320만 명 × 연간 12개 지원 = 연간 3,840만 건의 자소서 시장.**
SelfPR은 이 시장의 1%만 점유해도 연 115억 원 매출을 달성할 수 있다.
